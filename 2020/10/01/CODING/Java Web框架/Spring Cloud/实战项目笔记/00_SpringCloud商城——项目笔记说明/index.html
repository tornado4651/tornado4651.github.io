00_SpringCloud商城——实战项目系列笔记说明

发表于 2020-10-01 | 分类于 CODING ， Java Web框架 ， SpringCloud ， 实战项目笔记 | | 阅读次数: |

本篇文章主要说明我是如何学习这个商场实战项目的， 并对所有笔记的记录点等作出一个大概说明，就当作是一篇笔记的README吧～

  • 一、01_环境搭建与配置部分
  • 二、前端Vue项目记录部分
  • 三、后端基础技术框架记录说明部分
  • 四、后端开发辅助技术记录部分
  • 五、后端开发规范部分

​ 本来最初是想将老师的实战课程完全记录下来，细致到每个知识点，每个操作，让我这个系列的笔记最终可以变成一篇不需要视频的文字版教程，方便自己今后查阅的同时，也方便大家借鉴和操作

​ 但是，到了中期发现有太多操作性不方便记录，即使通过不断的调整记录，能够达到一些清晰的记录说明效果，但是这样太浪费时间，进行了许多没意义，对学习毫无帮助的事情，比如一些逻辑代码的记录，前端编写与问题解决方法等。

​ 因此，我在中后期开始，将笔记进行改版，不再做「流水帐」方式的记录，而是将项目中用到的知识点与框架尽可能按照老师的操作流程，尽可能的结合实际进行记录，放弃大多数无意义且耗时的操作记录。这样一来，虽然损失了最初打算做一个系列「文字教程」的目的，但是为自己提高了学习效率，今后翻看也可以更快捷到找到需要的点。

​ 而整个笔记分类与内容说明调整在下面具体说明：

一、01_环境搭建与配置部分

​ 这一部分的笔记主要记录了整个项目初期搭建的全过程，并且具体内容没有删减与改动，还是希望能详细记录下搭建项目的主要操作流程与方法，毕竟这部分我自身十分薄弱，对很多细节没有太清楚的操作认知。

二、前端Vue项目记录部分

​ 这一部分主要记录一些前端框架——Vue的相关知识点：包括操作、概念、js基础、前端开发流程等，虽说不是一个全栈工程师，但是多会点基础，说不定哪天能用到呢，比如小问题的自己查看等。这样也不用麻烦同事了。

三、后端基础技术框架记录说明部分

​ 这部分是整个系列笔记的重点！我将所有用到的「基础」、「主要」的后端框架与工具全部记录在这里，所以篇幅将是最长的，所以以后不一定会拆分成上下两部分。

四、后端开发辅助技术记录部分

​ 这一个部分，我将商城项目的开发过程中用到的部分简短，无需深入理解的「工具框架」记录下来，方便之后出现同样问题，或者想要深入学习时，反回来查找继续深入。

五、后端开发规范部分

​ 这一个部分，与java web的项目框架关系不大，更多是java的基础记录，又很多底层知识点的遗忘导致某些简单问题又在实战项目中困扰我很久，因此记录下来，以便复习巩固。

-------------本文结束感谢您的阅读-------------
