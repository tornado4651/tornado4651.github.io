数据校验JSR303

1）简单校验：

  • 使用javax.validation.constraints包下的注解对bean的字段进行校验标注
  • 在controller的入参前使用@Value注解标明该入参进行校验
  • 在需要校验的入参后紧跟BindingResult result的对象，其中包含了该字段的所有校验结果信息

2）分组校验：在不同场景下调用同一个接口时校验规则有可能不同，因此使用分组校验区分。

  • 在校验注解中通过groups={xxxx.class}值指定校验的组。
  • 在controller的入参前使用Spring提供的@Validate(xxxxx.class)注解标注该接口入参所在的校验组别
  • 注意：默认没有指定分组的校验字段，在指定了分组的情况下不生效，即@Validated({xxxx.class.xxx.class})。只有在单纯校验的情况下生效，即@Validated。

3）自定义校验
