我理解的Spring框架

Spring框架给我一种对java应用的各方面功能的高度概括提炼，将各种复杂配置封装成框架底层，并添加进一些特殊管理方式和新思想，使得使用更加简便。

但是凡事具有两面性，框架的继承度越高，虽然使得入门简便了，但是灵活性大大降低，虽然各类功能的开发在一定程度上缓解了很大一部分的开发灵活性，但仍然需要有时修改底层源代码对Spring框架作出最适合自己的改变。
